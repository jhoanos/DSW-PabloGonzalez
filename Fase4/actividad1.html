Que es HTML

Tal vez para muchas personas el termino Html es muy familiar, esto debido a que la mayoria de paginas Web disponibles en internet estan realizadas con este lenguaje de programacion.


Sin embargo muchos se preguntaran ¿que es HTML?, segun Preston,P(2015) "HTML son simples archivos de texto. El texto es la manera más corriente de introducir datos en un ordenador o un navegador.".
HTML, es un lenguaje basado en etiquetas o instrucciones que hacen que el texto se pueda visualizar de diferentes maneras a travez de un navegador.
